Программа является компилятором для языка машины поста
Программа принимает 2 файла:
    Один файл с алгоритмом машины Поста
    Второй с лентой из нулей и едениц для той же машины поста
На выходе программы получается:
    файл с кодом машины Поста переведенным в под си
    и готовая программа исполняющая код первого выходного файла
